# workflo/entitydb

Database for entities defined with `defentity` from `workflo/macros`.


## Testing

1. Install [boot](http://boot-clj.com/)
1. Clone this repository
2. Run the tests:
   - `boot test` to run tests once
   - `boot watch test` to run tests continuously on changes
